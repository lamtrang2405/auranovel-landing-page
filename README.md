# AuraNovel landing page

Static, single-screen landing page for a novel reading app.

## Run locally

- Open `index.html` in your browser (double click), or
- Use any local web server (recommended so anchors/assets behave consistently).

Example with PowerShell (if you have Python installed):

```powershell
python -m http.server 5173
```

Then open `http://localhost:5173/`.

## Customize

- **Brand name / copy**: `index.html`
- **Colors / layout / animations**: `styles.css`
- **Book covers**: replace files in `assets/` (keep same names or update the `img src` paths)
